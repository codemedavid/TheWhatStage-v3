Echo editor — which layout?

How variables get discovered and how the preview is shown

A

Sidecar — picker beside textarea, preview below

Variables list on the right (collapsed groups by namespace). Click inserts at cursor. Live preview renders under the textarea using sample data. Validation: unknown {{tokens}} get underlined.

B

Inline @-mention autocomplete + collapsed preview

Typing {{ opens a dropdown of variables (like Notion/Slack). Preview is a collapsed <details> under the textarea, expand to see. Picker is hidden until you ask for it.

C

Split — editor left, live preview right (Messenger bubble)

Two-column section. Left: textarea + variable chips above it (click to insert). Right: a Messenger-style bubble that re-renders on every keystroke with sample data. Preview always visible.
