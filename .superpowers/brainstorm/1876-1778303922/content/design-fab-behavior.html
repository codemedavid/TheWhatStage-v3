Button labels

Each pill uses cta_label from the linked page if set, otherwise falls back to the page title. Kind icon prefix: 📅 for booking, 📋 for form/qualification.

Sheet details

  • Mobile: slides up from bottom, max-height 85dvh, overflows with scroll, drag-handle decoration at top
  • Desktop: centered overlay, max-w-lg, max-height 85dvh, same scroll behavior
  • Close: ✕ button in header OR clicking/tapping the backdrop
  • Body scroll: locked while sheet is open (overflow-hidden on body)
  • Animation: CSS transition — slide-up on mobile, fade+scale on desktop
